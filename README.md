# Caso-final-Predicting-Customer-Churn
Análisis y predicción de la deserción de clientes (churn) en QWE INC mediante modelos de probabilidad (logit, probit, MPL).
Predicción de Deserción de Clientes – QWE INC

## Descripción
Este proyecto tiene como objetivo analizar y predecir la deserción de clientes (churn) en la empresa QWE INC mediante modelos de probabilidad para una variable binaria.

## Objetivo
Identificar los factores que influyen en la deserción de clientes.
Estimar un modelo de probabilidad (Logit/Probit/MPL).
Evaluar el desempeño del modelo predictivo.
